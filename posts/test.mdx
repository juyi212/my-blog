방법은 간단합니다.

> `fields`는 정적 필드, `computedFields`는 동적 필드라고 생각하시면 됩니다.

```typescript
import { defineDocumentType } from 'contentlayer/source-files';

export const Post = defineDocumentType(() => ({
  // ....
  computedFields: {
    category: {
      type: 'string',
      resolve: (post) => {
        const [category] = post._raw.flattenedPath.split('/');

        return category;
      },
    },
  },
}));
```
